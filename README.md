# Online Shop Microservices – Monorepo

Dieses Repository vereint mehrere unabhängige Microservices zu einer zentralen, wartbaren Plattform für einen Online-Shop.

Die einzelnen Services wurden ursprünglich als eigenständige Repositories entwickelt und anschließend in dieses Monorepo zusammengeführt, wobei die vollständige Commit-Historie erhalten geblieben ist.

Ziel ist es, die Weiterentwicklung, Wartung und Architekturpflege in einer einheitlichen Struktur zu ermöglichen.

---

## Überblick

Die Plattform bildet ein vollständiges E-Commerce-System auf Basis einer Microservice-Architektur ab und besteht aus mehreren spezialisierten Diensten, die über ein zentrales API-Gateway angebunden sind.

Jeder Service ist weiterhin logisch getrennt und besitzt seine eigene interne Struktur und Dokumentation.
